New Club Design

August 21, 2018 by Jorge Zhang

Hi everyone! Today I wanted to show you a small change to the club cards that will hopefully make it easier for newer players to understand clubs. I’ve found that newer players tend to avoid clubs because they are confusing to understand. This aesthetic change hopes to make them more approachable.

This is the original design of a club card. One of the issues was that it was difficult to tell which box meant which effect. Several players have confused the yearly ability for the immediate ability, or accidentally used the Resume Point value as the amount of Friends to purchase it. For very new players, it was difficult to find out how much each club cost.

I think that as players get more used to the cards, these problems go away. However, to make it easier for a new player, I decided to rework the design a bit.

Now, the Friend cost is in a red circle, which draws a person’s attention to it quickly. It also has “costs 3 Friends” written explicitly so that someone who is playing for the first time can make the connection between the red circle with the number and their red Friends circle.

What do you think about these changes? Let me know in the comments below!
